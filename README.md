# Introdução

Feito por Felipe Biava Favarin, estudante de Ciência da Computação no Insituto Federal Catarinense Campus de Videira para a disciplina de Redes de Computadores - Parte 2.

## Descrição

Repositório contendo códigos implementados em linguagem Python 3.12 que usam da biblioteca sockets e outras auxiliares para implementação de recursos / funcionalidades que estejam relacionadas a disciplina de Redes de Computadores.

## Execução

Para usar esse repositório público faça em seu terminal:

```js
git clone https://github.com/felipebfava/Sockets-Redes2.git
```

Para executar cada implementação entre na pasta específica e execute no terminal:

```js
python .\<nome_do_arquivo>
```
